# Projeto DSList - Intesivão Java Spring by Devsuperior

## Descrição
<p align=justify>Este projeto foi desenvolvido durante o evento Intesivão Java Spring da Devsuperior. Trata-se de uma 
simples API onde é disponibilizado 5 endpoints para realizar ações como listar os games, listas de games e um endpoint específico para 
alterar a ordem da lista de games. Essa API possui 3 perfis: teste, dev e prod. <b>O perfil de prod foi omitido</b>. A API roda 
em ambiente local com H2 e em ambiente de dev com PostgreSQL via container Docker.
</p>

## Tecnologias e ferramentas utilizadas
- Java 17
- Maven 
- Eclipse ou Visual Studio Code
- Postman
- PostgreSQL
- PgAdmin
- H2
- Spring Boot
- Spring Data JPA
- Projections
- Docker

#### Script para validar endpoint list replacement
```sql
SELECT TB_BELONGING .*, TB_GAME.TITLE FROM TB_BELONGING
INNER JOIN TB_GAME ON TB_GAME.ID = TB_BELONGING.GAME_ID
WHERE LIST_ID = 2
ORDER BY POSITION
```
